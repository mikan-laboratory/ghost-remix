# Smooth Jazz

## Description

- Ghost as a headless CMS with Remix and Chakra UI

## Dependencies

- [NPM](https://www.npmjs.com/)
- [Docker](https://www.docker.com/)
- [Tilt](https://tilt.dev/)

## Development

```
tilt up
```

## Useful Commands

- Seed DB

```
prisma db seed
```

- Prisma DB GUI

```
prisma studio
```
